# Screener d'Actions Boursières 📈

Script Python qui récupère les prix d'actions en temps réel via l'API Alpha Vantage.
Alertes automatiques, suivi de portefeuille, export CSV.

---

## 🚀 Fonctionnalités

- Prix en temps réel pour n'importe quelle action
- Variation du jour (montant + pourcentage)
- Alertes automatiques si seuil dépassé
- Sauvegarde des données en CSV

---

## 🛠️ Technologies

- Python 3 · Pandas · Requests · Alpha Vantage API

---

## ⚙️ Installation

```bash
git clone https://github.com/yann-guibert/screener-actions
cd screener-actions
pip install requests pandas
```

Obtiens une clé API gratuite sur : https://www.alphavantage.co/support/#api-key  
Remplace `VOTRE_CLE_API` dans le fichier par ta clé.

```bash
python screener_actions.py
```
